## Decision Trees

### Vessel Classification

```
Is the opening narrower than the body?
├─ Yes → JAR (constricted lip)
└─ No → Is it for flowers?
    ├─ Yes → Is opening narrow (1-2 stems)?
    │   ├─ Yes + height < 6-8" → BUD VASE
    │   └─ Yes + taller → VASE (narrow neck)
    └─ No → BOWL (open form)
```

**Source:** `taxonomies/vessel-anatomy.md`

### Color Selection

```
Can you identify specific color?
├─ Yes → Use specific term (e.g., "celadon", "chestnut")
└─ No → Use fallback color (e.g., "green", "brown")
```

**Source:** `taxonomies/colors.md` §Usage Guidelines
